Những ngôn ngữ lập trình mà tôi đã học

C#

C# là một ngôn ngữ lập trình hướng đối tượng, được phát triển bởi Microsoft trong .NET Framework. Nó hỗ trợ tính kế thừa, đa hình, đóng gói và trừu tượng hóa, giúp xây dựng các ứng dụng mạnh mẽ cho Windows và web. C# cũng có khả năng làm việc với các dịch vụ đám mây và ứng dụng di động nhờ vào .NET Core và Xamarin.

Python

Python là ngôn ngữ lập trình cao cấp, dễ học và sử dụng, với cú pháp rõ ràng và dễ đọc. Nó hỗ trợ nhiều paradigms lập trình như hướng đối tượng, lập trình hàm và lập trình thủ tục. Python được sử dụng rộng rãi trong khoa học dữ liệu, web development, tự động hóa, trí tuệ nhân tạo và nhiều lĩnh vực khác nhờ vào thư viện phong phú và cộng đồng mạnh mẽ.

Java Script

JavaScript (JS) là một ngôn ngữ lập trình động, chủ yếu được sử dụng để phát triển các ứng dụng web tương tác. Nó cho phép điều khiển các yếu tố trên trang web như DOM, xử lý sự kiện và giao tiếp với máy chủ qua AJAX. JavaScript là ngôn ngữ phổ biến trong việc phát triển frontend, và hiện nay cũng được dùng trong backend nhờ Node.js.
